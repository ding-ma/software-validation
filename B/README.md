# Part B of the project 


## Setup and execution

To setup the environment
```
pipenv shell
pipenv install
```

To run the tests 

```
behave 
```

## Folder Directory

All the `.feature` files will be stored in the `/features` folder. 
The corresponding steps python test will be in the `/features/steps` folder.
